## Após o clone, instalar as dependências e pacotes
npm install
ng add @angular/material

## Buildando aplicação
npm run build

## Rodando aplicação com o Electron
npm run forge-start

## Gerando aplicativo para desktop, pode demorar até 10 minutos para terminar o processo.
npm run make
